## Task 1: Fake News Detection

This project trains a machine learning model to classify news articles as fake or real.

### Features
- Text preprocessing (cleaning, stopwords removal, lemmatization)
- TF-IDF feature extraction
- Logistic Regression and Naive Bayes models
- Model evaluation using Accuracy, Precision, Recall, and F1-score
- Model saving using Pickle

### Folder Structure

FakeNews/
- fake_news.py  
- fake_news_model.pkl  
- requirements.txt  
- dataset/
  - Fake.csv  
  - True.csv  

### How to Run

1. Activate virtual environment:
.venv\Scripts\activate


2. Navigate to project:


cd FakeNews


3. Install dependencies:


pip install -r requirements.txt


4. Run script:


python fake_news.py
