# IPL Exploratory Data Analysis (EDA)

## Project Description
This project performs Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset. The analysis includes data cleaning, identifying the most successful teams and players, analyzing factors contributing to wins, and creating visualizations to present insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## How to Run the Script
1. Ensure you have Python installed.
2. Install the required libraries using the following command:
   ```
   pip install pandas matplotlib seaborn
   ```
3. Run the script using:
   ```
   python ipl_eda.py
   ```
